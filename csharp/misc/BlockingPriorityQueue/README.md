# BlockingPriorityQueue

キューが空の場合に、一定時間待ち受ける優先度付きキューの実装実験。

## プロジェクトの準備

```sh
dotnet new console
```

## Build

```sh
dotnet build
```

## Run

```sh
dotnet run
```
