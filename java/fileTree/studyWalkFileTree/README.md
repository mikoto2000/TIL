# Study walk file tree

## Build

```sh
gradle build
```

## Run

### Main.java

```sh
java -jar build/libs/studyWalkFileTree-1.0.jar PATH_TO_DIRECTORY
```

### Glob.java

```sh
java -cp build/libs/studyWalkFileTree-1.0.jar jp.dip.oyasirazu.studywalkfiletree.Glob PATH_TO_DIRECTORY GLOB_PATTERN
```
